# BranchTox - bTox
Tox client with branch support features. Create branches like in forum!


## Build Deps
### Build c-toxcore
```
git clone https://github.com/TokTok/c-toxcore
cd c-toxcore
mkdir _build && cd _build
cmake ..
make
sudo make install
```
### OR 
```
sudo apt install libtoxcore2 libtoxcore-dev
```
